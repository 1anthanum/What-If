AI 辩论室 核心模块

多个 AI 角色围绕你的假设展开回合制辩论。每个角色有独立的身份、立场和语言风格。 你可以随时注入新事件打断辩论走向。

操作流程

1
输入假设
在场景输入框中填写你的 what-if 假设，例如「如果全球粮食产量突然翻三倍，会发生什么？」。系统会自动推荐 4-6 个相关角色。
2
观看辩论
每个角色依次发言，文字实时流式呈现。每张角色卡片左上角有颜色标识和角色图标。发言完成后会显示完整内容。
3
注入事件（可选）
点击「注入事件」按钮，输入突发事件（如「旱灾来了」）。所有角色在下一回合必须回应这个新变量。
4
查看分析摘要
点击「分析师摘要」查看 AI 生成的结构化总结：共识点、核心分歧、潜在风险。

💡
技巧：在辩论进入第 3-4 回合后注入事件效果最好——此时各角色已建立初步立场，新变量能产生更有戏剧性的转折。

因果图谱 核心模块

AI 将你的假设分解为因果网络：哪些变量影响哪些变量、影响是正向还是负向。 你可以"冲击"任何节点，观察级联效应沿因果链传播。

操作流程

1
生成因果图
输入场景后，AI 自动识别 10-20 个关键变量并建立因果关系。等待 SSE 流完成后，D3 力导向图自动渲染。
2
浏览图谱
拖拽节点调整布局，滚轮缩放。节点颜色代表类别：●经济 ●社会 ●环境 ●政治。线条颜色：绿=正向、赭=负向、蓝虚线=复杂关系。
3
冲击节点
点击任意节点 → 侧边栏弹出 → 输入扰动描述（如「粮食价格下降 50%」）→ 点击「分析传播」。
4
查看传播结果
受影响的节点会高亮，侧边栏显示每个节点的 AI 评估（影响程度、传播延迟、可能的反馈环）。

历史反事实 核心模块

选择一个历史事件，修改关键参数，AI 为你生成"如果当年不是那样"的替代时间线。 支持单一时间线、Ensemble 探索（多分支）、证伪引擎和用户知识注入。

四种模式

A
单一时间线
选择事件 → 修改假设 → 生成一条反事实时间线。每个时间点可展开查看 AI 的推理过程。可对生成的时间线运行「证伪分析」或「标注修正」。
B
可能性探索 (Ensemble)
切换到「探索模式」→ 15 个快速模型并行发散 → 聚类为 3-4 个叙事方向 → 每个方向精炼为完整时间线。结果展示为扇形分支图，分支宽度反映共识度。
C
吸引子检测
输入 3-5 个不同的修改假设 → 对每个假设运行 Ensemble → 跨所有结果寻找「无论怎么改都会发生」的收敛结局。
D
具身视角
用历史真实人物（非抽象视角）作为探索代理。选择人物 → 每人按自身世界观探索 → 按「利益联盟」而非「叙事主题」聚类。
  Ensemble 三阶段管线

  ┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
  │ Stage 1: 发散    │    │ Stage 2: 聚类   │    │ Stage 3: 精炼   │
  │                 │    │                 │    │                 │
  │ Haiku × 15     │───▶│ Sonnet × 1      │───▶│ Sonnet × 3-4   │
  │ 并行短调用     │    │ 语义聚类        │    │ 完整时间线      │
  │ ~300 tok/条    │    │ 分成 3-4 簇     │    │ ~2000 tok/条   │
  │ 成本 ≈ $0.003  │    │ 成本 ≈ $0.01    │    │ 成本 ≈ $0.09   │
  └─────────────────┘    └─────────────────┘    └─────────────────┘

  总成本：≈ $0.10–0.15（与单次生成相当）
💡
证伪引擎：生成时间线后，点击「运行证伪分析」。AI 将以对抗性视角攻击时间线中最弱的因果推理，帮你识别哪些推断最不可靠。脆弱性评分 0-1，颜色从绿到红。

闭环推演 核心模块

将多个模块串联成循环：反事实 → 因果图 → 辩论 → 反馈修正，或启动自主循环让 AI 持续迭代哲学问题。

两种模式

  模式 A：反馈循环

     反事实时间线 ──▶ 因果图谱 ──▶ AI 辩论 ──┐
          ▲                                    │
          └──── 辩论结论作为约束 ◀─────────────┘
                  (最多 3 轮迭代)

  模式 B：自主哲学循环

     问题输入 ──▶ 5 个 Persona 辩论 ──▶ 综合 ──▶ 提取子问题 ──┐
       ▲                                                        │
       └────────────── 子问题作为新输入 ◀──────────────────────┘
                       (最多 10 轮，可收敛)

自主循环操作

1
选择模式
在闭环推演标签页中，切换到「自主探索」模式。选择「历史」或「哲学」子模式。
2
配置参数
设置种子假设、最大循环数。哲学模式下可启用高级选项（对抗测试、分歧热力图、决策分支树）。
3
启动并观察
点击启动后，系统自主运行。演化链实时显示假设如何从初始问题逐步深化。可随时取消。

认知分歧热力图 高级功能

可视化各哲学 Persona 对辩论核心论点的立场差异。红色=反对，蓝色=支持，灰色=中立。 帮你一眼看清谁同意谁、分歧在哪里。

启用方式

在自主循环配置中，开启 认知分歧提取 开关。系统会在每轮辩论后自动提取立场矩阵。

对抗性压力测试 高级功能

引入「魔鬼代言人」角色，专门攻击辩论中最弱的论点。 它会看到所有其他 Persona 的完整发言后再发动精准攻击。

启用方式

在自主循环配置的高级选项中，开启 对抗性测试 开关（红色按钮）。 

  两阶段辩论流程

  ┌──── Phase 1：正常辩论 ─────────────────────────────┐
  │                                                     │
  │  理性主义 → 存在主义 → 实用主义 → 东方哲学  │
  │  (4 个 Persona 依次发言)                            │
  │                                                     │
  └─────────────────────┬───────────────────────────────┘
                        │ 所有发言汇总
                        ▼
  ┌──── Phase 2：对抗攻击 ─────────────────────────────┐
  │                                                     │
  │  ⚡ 魔鬼代言人                                   │
  │  看到全部 Phase 1 内容，找出最弱论点               │
  │  给出 1-5 脆弱性评分 + 攻击向量                    │
  │                                                     │
  └─────────────────────────────────────────────────────┘
⚡
魔鬼代言人的价值：它不是简单地「唱反调」，而是有策略地攻击——找到论证链中最脆弱的环节，提供反例，并评估如果这个论点被推翻会产生的连锁影响。

决策分支树 可视化

每轮循环后，AI 不仅选择一个子问题继续，还展示未被选中的替代方向。 你可以看到「如果当时走了另一条路，会探索什么问题」。

启用方式

在高级选项中开启 决策分支 开关。每轮会生成 3 个正交性最大化的候选问题。

未探索分支的价值在于：它们揭示了你当前问题空间中「你没有问但值得问的方向」。这些分支可以成为你下一次探索的种子假设。
